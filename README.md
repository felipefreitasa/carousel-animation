# Carousel animation 

## Install 

- First, make sure you have Expo CLI installed: `npm install -g expo-cli`

- Install dependencies: `npm i` or `npm install`

- Run Project Locally: `npm start`

## Technologies

- React Native
- Animated API
- FlatList
